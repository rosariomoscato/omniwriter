- status_notes TEXT DEFAULT ''

#### 2. Backend AI Prompt Updates (server/src/routes/projects.ts)

- Added "status_notes" field for detailed notes about final state
- Instructions to deduce final state from text

#### 3. Interface Updates (server/src/routes/projects.ts)
ParsedCharacter interface - Added optional fields:
- status_at_end?: string
- status_notes?: string

#### 4. Response Parsing (server/src/routes/projects.ts)

- Extracts status_notes with default ''
