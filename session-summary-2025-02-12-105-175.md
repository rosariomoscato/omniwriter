## Toast System Features

### Visual Design
- **Success:** Green border-left, check icon, light green background
- **Error:** Red border-left, alert icon, light red background
- **Info:** Blue border-left, info icon, light blue background
- **Warning:** Amber border-left, warning icon, light amber background

### Dark Mode Support
- All toast types have dark mode variants
- Dark backgrounds use `/20` opacity for subtle appearance
- Icons retain their colors in dark mode

### Accessibility
- `aria-live="polite"` - announces toasts to screen readers
- `aria-atomic="true"` - reads toast as single unit
- `role="alert"` - marks content as alert
- `aria-label` - dynamic labels with type and message
- Escape key support - dismisses all toasts
- Keyboard-navigable close buttons

### User Experience
- **Auto-dismiss:** 5 seconds default, configurable per toast
- **Manual dismiss:** Click close button or press Escape
- **Stacking:** Newest toasts at top, vertical layout
- **Limit:** 3 toasts max (prevents screen overflow)
- **Animation:** Smooth slide-in from right
