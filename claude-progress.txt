### Feature #389: Bug: Lunghezza capitoli ancora non rispettata nonostante feature #388

**Objective:** La feature #388 risultava implementata e passing, MA l'utente stava ancora ottenendo solo ~1145 parole invece delle 3000 parole attese con l'opzione "Lungo".

**Root Cause Analysis:**
The bug was in all AI provider implementations (`getDefaultModel()` method). When a user configured their own model (e.g., "openai/gpt-5.2-pro"), the provider was ignoring it and always using the hardcoded default model (e.g., "gpt-4o-mini").

**Root Cause:**
In all provider files, `getDefaultModel()` returned only the hardcoded `PROVIDER.DEFAULT_MODEL`:
```typescript
getDefaultModel(): string {
  return Provider.DEFAULT_MODEL;  // BUG: Ignores this.config.defaultModel
}
```

This meant that even though the user's selected model (`openai/gpt-5.2-pro`) was correctly loaded from the database and stored in `this.config.defaultModel`, it was never used!

**Fix Applied:**
Updated `getDefaultModel()` in all 5 AI providers to check `this.config.defaultModel` first:

```typescript
getDefaultModel(): string {
  // Feature #389: Use configured default model if available
  return this.config.defaultModel || Provider.DEFAULT_MODEL;
}
```

1. `server/src/services/ai/providers/OpenAIProvider.ts` - Fixed + added debug logging

1. Create a new table `export_history_new` with the updated constraint

function truncateToWordLimit(content: string, maxWords: number): { content: string; wasTruncated: boolean } {
  const words = content.split(/\s+/).filter(w => w.length > 0);
  if (words.length <= maxWords) {
    return { content, wasTruncated: false };
  }

  // Truncate to maxWords and add ellipsis
  const truncatedWords = words.slice(0, maxWords);
  return {
    content: truncatedWords.join(' ') + '...',
    wasTruncated: true
  };
}

3. No exclusion patterns for recovery/survival after violent context
4. Patterns used [^.!?]* which didn't work across sentences

**Fix Applied:**

2. Changed priority order: ALIVE > INJURED > DEAD (was DEAD > INJURED > ALIVE)

4. Added explicit death keywords: Must have clear confirmation like 'mori', 'cadavere di', 'non si rialzo piu'
5. Added alive keywords: 'sopravvive', 'si riprese', 'celebro', 'vittoria', etc.
