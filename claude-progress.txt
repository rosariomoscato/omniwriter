**Updated Forms:**

- ProjectDetail forms (chapter/character/source) clear state properly
- All submit buttons disable during loading

**How it Works:**
1. User submits form
2. On success: form data cleared, navigation uses replace: true
3. User cannot navigate back to pre-filled form
4. If user presses back, form is empty - resubmitting creates new entry (not duplicate)
