### Feature #232: Complete AI Chapter Generation System with Streaming

**Objective:** Implementare un sistema completo di generazione AI nell'editor dei capitoli con streaming SSE, progress indicator, cancellazione e supporto Human Model.

**Implementation:**

#### 1. Backend SSE Endpoint (server/src/routes/chapters.ts)
- Added `POST /api/chapters/:id/generate-stream` endpoint
- Fetches chapter context: characters, locations, plot events, sources
- Builds localized system prompt (Italian/English) based on user preference
- Integrates Human Model style profile when provided
- Streams AI-generated content via Server-Sent Events
- Sends phase updates: structure → writing → revision
- Falls back to template generation when AI provider unavailable
- Updates chapter in database after generation
- Logs generation with token usage tracking

#### 2. Frontend API Streaming Method (client/src/services/api.ts)
- Added `generateChapterStream()` method with callbacks:
  - `onPhase(phase, message)` - Phase change notifications
  - `onDelta(content)` - Streaming content chunks
  - `onDone(data)` - Generation completion
  - `onError(error)` - Error handling
- Returns `{ abort, promise }` for cancellation support
- Uses native EventSource parsing via fetch ReadableStream

#### 3. ChapterEditor UI Updates (client/src/pages/ChapterEditor.tsx)
- Added state variables:
  - `isGenerating` - Generation in progress flag
  - `generationPhase` - Current phase (structure/writing/revision)
  - `generationMessage` - Phase status message
  - `humanModels` - Available Human Models list
  - `selectedHumanModelId` - Selected model for styled generation
  - `showHumanModelDropdown` - Dropdown visibility
  - `generationAbortController` - For cancellation
  - `streamingContent` - Accumulated generated content

- Added useEffect to load Human Models on mount
- Added `handleGenerateChapter(useHumanModel)` function
- Added `handleCancelGeneration()` function

- Added UI Components:
  - **Generate Button** (purple): Standard AI generation
  - **With Style Button** (amber): Human Model generation
  - **Human Model Dropdown**: Select from available models
  - **Progress Indicator**: Shows phase and message during generation
  - **Cancel Button**: Interrupt ongoing generation

#### 4. Translations (client/src/i18n/locales/)
- Added `chapterEditor.generation.*` keys in it.json and en.json:
  - `generate`, `generateStandard`, `generateWithStyle`
  - `selectModel`, `noHumanModels`
  - `analyzing`, `writing`, `revision`
  - `success`, `error`, `cancel`, `cancelled`
  - `structure`, `writing_phase`, `revision_phase`

- Uses blue/info theme for regeneration (non-destructive action)
